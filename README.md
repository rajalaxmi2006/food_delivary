# 🍔 Food Delivery Landing Page

A modern and responsive **Food Delivery Website UI** built using HTML and CSS. This project showcases a clean layout, attractive design, and user-friendly interface for a food ordering platform.

---

## 🚀 Live Demo
👉 https://your-username.github.io/food-delivery/

---

## 📌 Features

- 🍽️ Beautiful hero section with call-to-action  
- 🛵 Service section explaining how it works  
- 🍕 Menu showcase with food items and pricing  
- 📬 Newsletter / contact section  
- 📱 Fully responsive design  
- 🎨 Clean and modern UI  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox + Grid)  
- Google Fonts (Poppins)
